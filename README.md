# React 19 + Vite + TypeScript Template

Starter template for projects with:

- React 19, Vite, TypeScript
- Tailwind CSS v4 (with @tailwindcss/vite)
- shadcn/ui primitives (via cva/clsx utilities)
- lucide-react icons
- TanStack Query
- SWC (React SWC plugin)

## Quick Start

1. Install: `pnpm i` or `npm i` or `yarn`
2. Dev: `npm run dev`
3. Build: `npm run build`
4. Preview: `npm run preview`
5. Lint: `npm run lint`
